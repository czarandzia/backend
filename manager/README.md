## Opis projektu
REST API do zarządzania zadaniami (Task Manager) stworzone na zajęciach z backend development. API umożliwia tworzenie, odczyt, aktualizację i usuwanie zadań. Dane są przechowywane w pliku JSON.

## Technologie
- [JavaScript]
- [Express]
- [Node.js File System (fs)]

## Endpointy
1. GET /health - status API
2. GET /tasks - pobranie wszystkich zadań
3. GET /tasks/:id - pobranie konkretnego zadania
4. POST /tasks - dodanie nowego zadania
5. PUT /tasks/:id - aktualizacja zadania
6. DELETE /tasks/:id - usunięcie zadania

## Testy
- API testowane za pomocą programu postman
- wszystkie endpointy działają poprawnie
